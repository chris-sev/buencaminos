1. HTML files and tructure

This theme is a responsive layout with one to four columns. The general template structure is the same throughout the template.

<!-- ========== BANNER START ========== -->

<div id="slides">
    <div class="slides-container">

        ## BANNER IMAGES GO HERE ##

    </div>
    <div class="tint">
        <div class="welcome text-center">

            ## SLOGAN GOES HERE ##

        </div>
    </div>
</div>

<!-- ========== BANNER END ========== -->

<!-- ========== MENU START ========== -->

<div class="navbar navbar-default navbar-fixed-top">
    <div class="container">

        </div>

<!-- ========== MENU END ========== -->

## MAIN CONTENT GOES HERE ##

</footer>

<!-- ========== FOOTER END ========== -->

1.1. Navigation

Site navigation is found in <ul class="nav navbar-nav"> inside .navbar div. To create main menu items, add list elements to <ul class="nav navbar-nav">. You can add sub menu items to the main menu items adding an ul tag with list elements inside. You also have to add the parent class to the parent li elements. Adding current class to list elements makes the menu item selected/highlighted.

<ul class="nav navbar-nav navbar-right">
    <li><a href="#about">About Us</a></li>
    <li><a href="#story">Our Story</a></li>
    <li class="dropdown">
        <a href="#" class="dropdown-toggle" data-toggle="dropdown">People <b class="caret"></b></a>
        <ul class="dropdown-menu">
            <li><a href="index.html#people">Most Important People</a></li>
            <li><a href="person1.html">Person Profile Page 1</a></li>
            <li><a href="person2.html">Person Profile Page 2</a></li>
        </ul>
    </li>
    <li><a href="#events">Events</a></li>
    <li><a href="#accommodation">Accommodation</a></li>
    <li class="dropdown">
        <a href="#" class="dropdown-toggle" data-toggle="dropdown">Blog <b class="caret"></b></a>
        <ul class="dropdown-menu">
            <li><a href="index.html#blog">Home Page Blog</a></li>
            <li><a href="blog.html">Blog Post List</a></li>
            <li><a href="post.html">Blog Single Post</a></li>
        </ul>
    </li>
    <li><a href="#gallery">Gallery</a></li>
    <li><a href="#rsvp">RSVP</a></li>
</ul>

2. CSS files and stucture

The following css files are used in every template html file. They can be found in the HEAD section of the page. Style.css and bootstrap.min.css are the template css files, the other ones are the Google fonts, Owl Carsouel, Fancybox and Awesome Font icons css files.

<head>

    <!-- Pace Loader -->
    <link href="css/pace.css" rel="stylesheet">

    <!-- Bootstrap -->
    <link href="css/bootstrap.css" rel="stylesheet">

    <!-- Custom CSS -->
    <link href="css/style.css" rel="stylesheet">

    <!-- Superslides CSS -->
    <link href="css/superslides.css" rel="stylesheet">

    <!-- Countdown CSS -->
    <link href="css/countdown.css" rel="stylesheet">

    <!-- Owl Carousel stylesheet -->
    <link href="css/owl.carousel.css" rel="stylesheet">

    <!-- Owl Carousel Default Theme -->
    <link href="css/owl.theme.css" rel="stylesheet">

    <!-- Fancybox -->
    <link href="css/jquery.fancybox.css" rel="stylesheet">

    <!-- Google Fonts -->
    <link href="http://fonts.googleapis.com/css?family=PT+Serif%7CEngagement%7COpen+Sans:800" rel="stylesheet">

    <!-- Font Awesome Icons -->
    <link href="http://netdna.bootstrapcdn.com/font-awesome/4.0.2/css/font-awesome.css" rel="stylesheet">

</head>

2.1. Change Google font

Google font can be easily replaced in 2 simple steps:

  1. Edit the template file.
  2. Replace the css file for PT Serif font with yours (e.g. "PT Sans"), as you can see below.
<head>
    ...
    <!-- Google Fonts -->
    <link href="http://fonts.googleapis.com/css?family=PT+Serif%7CEngagement%7COpen+Sans:800" rel="stylesheet">
    ...
</head>

4. Javascript files and structure

This template requires the following javascript files. You should place them at the end of every template html file, just before the closing </body> tag. All the script and plugin settings are in two files - "settings-home.js" and "settings-inner.js"

<script src="js/pace.min.js"></script>

<!-- Modernizr Plugin -->
<script src="js/modernizr.custom.97074.js"></script>

This template uses Google Maps. You can set multiple custom loacations, pin images, map color and many more things.
Click here for more info and full documentation

    {
      "stylers": [
        { "saturation": -100 },
        { "gamma": 1 }

      "featureType": "water",

        { "lightness": -12 }
